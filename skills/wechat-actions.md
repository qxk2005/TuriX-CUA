# WeChat Actions

## Core workflow
- Open WeChat first if the user did not open it. 
  - You can press `Command+Space` to trigger Spotlight, type `WeChat` and hit `Enter` to open/activate it.
  - Or, if you see the WeChat icon in the Dock or on screen, click it.
- Once WeChat is open and active:
  - Locate and click the WeChat Search bar (usually on the top left of the WeChat window) or press `Command+F` to focus on the search field.
  - Type the target contact name, such as `文件传输助手` (File Transfer Assistant) or the specific English name if the user system is set to English.
  - Press `Enter` or click on the search result to open the chat room with that contact.
- Locate the message input box at the bottom of the active chat window:
  - Click on the input area to make it focused.
  - Type the desired message (e.g. `hello`).
  - Press `Enter` or click the Send button to send the message.

## Verification
- Confirm that the message you typed appears in the chat message history container above the input area.
